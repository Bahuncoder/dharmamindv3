## Performance Considerations

- **Detection Rate**: ~95% for clear, well-lit poses
- **Processing Speed**: ~30 FPS on modern hardware
- **Memory Usage**: ~500MB for full system
- **Model Size**: ~50MB for trained classifier

### Optimization Tips

1. **Lighting**: Ensure good, even lighting
2. **Background**: Use plain backgrounds for better detection
3. **Clothing**: Wear fitted clothing for accurate pose estimation
4. **Distance**: Position 6-8 feet from camera
5. **Angle**: Front-facing poses work best

## Troubleshooting

### Common Issues

**1. MediaPipe Installation Errors**

```bash
# Try specific version
pip install mediapipe==0.10.0

# Or use conda
conda install -c conda-forge mediapipe
```

**2. Low Detection Confidence**

- Improve lighting conditions
- Ensure full body is visible
- Remove background distractions
- Adjust detection_confidence parameter

**3. Incorrect Classifications**

- Retrain model with more data
- Adjust classification_confidence threshold
- Ensure proper pose execution

**4. Performance Issues**

- Reduce max_frames_per_second
- Lower model_complexity
- Use GPU acceleration if available
